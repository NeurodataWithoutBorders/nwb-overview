EEGLAB
------

.. short_description_start

:ref:`analysistools-eeglab` is a MATLAB-based, time series visualizer for Electrocorticography (ECoG/iEEG/EEG) signals stored in NWB files. After installing the NWB-IO EEGLAB plugins, time series, channel information, spike timing, and event information may be imported, processed, and vizualised using the EEGLAB graphical interface or scripts. NWB file may also be exported.
